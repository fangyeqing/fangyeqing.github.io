git环境

配置github

https://github.com/

新建repository

点击New repository

Repository name下填写博客目录，例如：username.github.io
Description下填写描述，例如：我的博客

生成github page

创建目录后进入该目录，“Setting”进行设置。

下拉到“GitHub Pages”模块，点击“Launch automatic page generator”按钮，生成github page。

刚才配置的“username.github.io”已经可以访问了。
